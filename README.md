# WinController
windows 远程管理：屏幕监控、键盘记录、文件管理、命令执行

### 项目仍在开发中... 进度 8/10
#### 目前可以使用的功能：
    屏幕监控、键盘记录、文件管理（上传、遍历、在线编辑、删除、重命名、下载）、命令执行

### 客户端构建可执行文件时创建新的环境 进入虚拟环境后执行
    ` (venv) E:\xxxxxx\WinController\client>build.cmd`
### 当然，你也可以自己编译，如果有点基础的话
